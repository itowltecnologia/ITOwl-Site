# Repositório de Logotipos ITOWL

Colete aqui os arquivos de imagem dos logotipos da ITOWL. 

Para utilizar os logotipos no site de forma automática, recomendamos que você os renomeie ou os atualize nos componentes React. Exemplo de arquivos sugeridos:

- `logo-white.png` (para a barra de navegação transparente e rodapé escuro)
- `logo-blue.png` (para usar em fundos claros)
- `logo-black.png`

**Como fazer o upload:**
Use o explorador de arquivos ao lado para arrastar e soltar as imagens que você acabou de enviar diretamente para dentro desta pasta `/public/logos`.
